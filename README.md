# Intelligent Wrong-Way Vehicle Detection System with Real-Time Warning and Auto Fine

##  Project Overview
The Intelligent Wrong-Way Vehicle Detection System is a smart road safety solution designed to detect vehicles traveling in the wrong direction on highways, flyovers, and one-way roads. Wrong-way driving is a major cause of serious road accidents and fatalities.

This system uses Artificial Intelligence, Computer Vision, and IoT technologies to monitor traffic in real time and automatically detect vehicles moving against the designated traffic flow.

Once a violation is detected, the system immediately alerts the driver and traffic authorities, reducing accident risks and improving road safety.

---

##  Objectives
- Detect wrong-way vehicles automatically
- Reduce road accidents caused by wrong-direction driving
- Provide real-time alerts to drivers
- Automatically generate fines for traffic violations

---

##  Technologies Used
- Computer Vision
- Image Processing
- Machine Learning
- Edge AI
- Cloud Computing
- Automatic Number Plate Recognition (ANPR)

---

##  Key Features
- Real-time vehicle movement monitoring
- AI-based direction detection
- Automatic number plate recognition
- Instant warning alerts through digital signboards
- Automatic e-challan generation
- Violation record storage in cloud database
- Analytics dashboard for traffic authorities

---

##  System Workflow
1. Cameras monitor road traffic continuously.
2. AI model detects vehicles and identifies their direction.
3. If a vehicle moves in the wrong direction:
   - Warning alert is sent.
   - License plate is captured.
   - Automatic fine is generated.
4. Data is stored in a centralized cloud database.

##  Expected Impact
